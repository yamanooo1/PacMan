# PacMan Game
Yaman Haj Ahmad
s0241093

A C++ implementation of PacMan using SFML, featuring ghost AI, level progression, and score tracking.

## Features

- 4 distinct ghost AI personalities (Random, Chaser, Ambusher)
- Exponential score decay system
- Fear mode with ghost vulnerability
- Level progression with increasing difficulty
- Death animations and sound effects
- High score leaderboard (top 5)
- Smooth grid-based movement

## Requirements

- **C++17** or higher
- **CMake 3.28** or higher
- **SFML 2.6** (graphics, window, system, audio)

### Installing SFML

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install libsfml-dev
```
